# Python组包和拆包

- 概述：
	- 组包：`n => 1`的过程，例如：`list1 = [10, 20, 30]`
	- 拆包：`1 => n`的过程，例如：`a, b, c = list1`

- 代码演示：

	```python
	list1 = [10, 20, 30]	# 组包
	print(list1)	# [10, 20, 30]
	
	# 对列表拆包
	a, b, c = list1		# 拆包
	print(a, b, c)		# 10 20 30
	
	# 对元组拆包
	s1, s2 = ('张三', '李四')
	print(s1, s2)		# 张三 李四
	
	# 对字典拆包，注意：只能获取键（因为：键具有唯一性，而值会重复）
	k1, k2 = {'name': '李白', 'age': 26}
	print(k1, k2)		# name age
	```
